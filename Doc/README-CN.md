![example](./example.png)

## 已验证平台
- PC(Win) Standalone
- WebGL

## 开发环境
- Windows 10
- Unity2020.3.30f1c1

## 如何使用

- 导入自己的模型:

  - 在这里添加模型，并填入路径（本地文件路径或网址）:

    ![add_model](./add_model.png)

  - 在这里添加作为目标的方块，并填入名字和跳转网址：

    ![add_target](./add_target.png)

- 改变小地图:

  运行后利用截图脚本截图，并把截图替换到Quad中：

  ![image-20220316142040365](./minimap.png)
